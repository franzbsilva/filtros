# filtrosemjava
Aplicação Java que aplica filtros em imagens.  
Foi desenvolvida como requisito para obtenção de nota de em disciplina de computação grafica 1 os filtros são: 
- Escala de Cinza 
- Negativo
- Limiarização (Threshold) 
- Colorização de Imagem 
- Sépia
